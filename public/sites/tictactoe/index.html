Tic Tac Toe

Introduction

Welcome to Neural-Network Tic Tac Toe. This neural network's training set is created using a tic-tac-toe minimax algorithm and a randomly moving player. This project was created by Brian Hoy for the 2016 MTHS Science Fair. You can either run the automated test, which will do 6 trials and print the results to the console (may take some time), or train a neural network and play against it.

This project uses the Brain library by Heather Arthur and a minimax tic tac toe algorithm by Richard To.
